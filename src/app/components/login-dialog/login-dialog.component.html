Power BI Token

Please enter your Power BI AAD token
In order to obtain a token please do the following:
  • Open Powershell as administrator
  • Run (once) the command Install-Module -Name MicrosoftPowerBIMgmt
  • Run the command Login-PowerBIServiceAccount
  • Run the command Get-PowerBIAccessToken -AsString
  • Copy the token and paste it here
